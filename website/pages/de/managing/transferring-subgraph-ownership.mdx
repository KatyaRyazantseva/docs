---
title: Übertragung des Eigentums an Untergraphen ( Subgraph )
---

The Graph unterstützt die Übertragung des Eigentums an einem Subgraphen. Wenn Sie einen Subgraphen im Mainnet bereitstellen, wird eine NFT auf die Adresse geprägt, die den Subgraphen bereitgestellt hat. Die NFT basiert auf einem Standard ERC721, so dass sie leicht auf verschiedene Konten übertragen werden kann. Derjenige, der die NFT besitzt, kontrolliert den Subgraphen. Wenn der Eigentümer beschließt, die NFT zu verkaufen oder zu übertragen, kann er keine Änderungen oder Aktualisierungen an diesem Subgraphen im Netzwerk mehr vornehmen. Diese Funktion bietet nicht nur mehr Flexibilität im Entwicklungszyklus, sondern erleichtert auch bestimmte Anwendungsfälle, wie z. B. die Übertragung der Kontrolle auf eine Multisig oder die Erstellung durch ein Community-Mitglied im Namen einer DAO.

Wenn Sie einen Subgraph im Mainnet bereitstellen, wird eine NFT auf die Adresse gemint, die den Subgraph bereitgestellt hat. Die NFT basiert auf einem Standard ERC721, sodass sie leicht auf verschiedene Konten übertragen werden kann.

Wer den NFT besitzt, kontrolliert den Teilgraphen. Wenn der Eigentümer beschließt, das NFT zu verkaufen oder zu übertragen, kann er keine Änderungen oder Aktualisierungen an diesem Teilgraphen im Netzwerk mehr vornehmen.

Diese Funktionalität erhöht nicht nur die Flexibilität im Entwicklungszyklus, sondern macht auch bestimmte Anwendungsfälle bequemer, z. B. das Verschieben Ihrer Kontrolle zu einer Multisig oder die Erstellung durch ein Community-Mitglied im Namen einer DAO.

## Betrachtung Ihres Subgraphen als NFT

Um Ihren Teilgraphen als NFT zu sehen, können Sie einen NFT-Marktplatz wie OpenSea besuchen:

```
https://opensea.io/your-wallet-address
```

Oder ein Wallet-Explorer wie **Rainbow.me**:

```
https://rainbow.me/your-wallet-addres
```
